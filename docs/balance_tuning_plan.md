# Balance Tuning Plan (MVP)

## 목표
- 평균 세션 길이: 4~7분
- 2048 클리어 체감: 숙련 20~35%, 캐주얼 5~12%
- 1회 플레이 포인트 수급: 25~60pt (클리어 시 150~220pt 체감)

## 관측 지표
- 평균 점수/최고 점수 분포
- 클리어율, 1024 도달율
- 아이템 사용 빈도/재구매율
- 미니게임 참여율/버프 획득률

## 튜닝 레버 (코드 상수)
- SCORE_TO_POINTS (기본 120)
- CLEAR_BONUS (기본 150)
- STREAK_BONUS_RATE (기본 0.3)
- STREAK_MAX (기본 3)
- SPAWN_RATE_4 (기본 0.1)
- MINIGAME_MIN_POINTS / MAX_POINTS / BUFF_THRESHOLD
- ITEM_DEFS 가격 및 한도

## 실험 플랜 (초기)
1. 포인트 수급
   - A: SCORE_TO_POINTS=120 (기본)
   - B: SCORE_TO_POINTS=100 (보상 상향)
2. 미니게임 보상
   - A: 50~120 (기본)
   - B: 60~140 (보상 상향)
3. 아이템 접근성
   - A: 가격 유지
   - B: 가격 -10% (입문 완화)

## 진행 절차
1. 1차 내부 플레이 20~30판 기록
2. 지표 요약 후 레버 조정
3. 2차 테스트로 체감 비교

## 기록 포맷 (권장)
- 날짜 / 플레이어 수준 / 평균 점수 / 평균 포인트 / 클리어율 / 사용 아이템
