# Forest Merge 2048 - Test Checklist (MVP)

## Automated
1. `npm run test` -> engine unit tests pass.
2. `npm run build` -> production build passes.

## Manual smoke tests
1. `npm run dev` 후 브라우저 접속.
2. Splash -> Lobby 전환되는지 확인.
3. Lobby에서 포인트/연속클리어/버프 표시 확인.
4. Upgrades 버튼으로 업그레이드 화면 진입/복귀 확인.
5. 업그레이드 구매 시 레벨/포인트/다음 비용 갱신 확인.
6. Start Game 클릭 시 보드에 타일 2개 생성.
7. 스와이프/방향키로 이동되는지 확인.
8. 같은 값이 합쳐질 때 점수 증가 및 콤보 증가.
9. 1024 도달 시 긴장 UI 표시.
10. 2048 도달 시 Clear 오버레이 표시.
11. 더 이상 이동 불가 시 Game Over 오버레이 표시.
12. Result 버튼 -> 결과/보상/상점 화면 이동.
13. 광고 보상 버튼 클릭 시 포인트 추가 (1회 제한).
14. 아이템 구매 후 인벤토리 증가 확인.
15. 게임 화면 아이템 사용: Undo, Shuffle, Hammer, Double Score.
16. 아이템 사용 제한(게임당 횟수) 적용 확인.
17. 미니게임 30초 종료 후 보상 지급 및 버프 표시.
18. Back to Lobby 동작 확인.
19. 업그레이드 적용 후 점수/콤보 보너스/종료 보너스가 소폭 증가하는지 확인.

## Ranking + Firestore QA
1. 랭킹 화면에서 탭 전환(일간/주간/월간/전체) 동작 확인.
2. `내 최고 기록` 카드가 별도로 표시되는지 확인.
3. Top 1~3이 일반 행과 다른 강조 스타일로 표시되는지 확인.
4. 동일 계정으로 여러 판 플레이 후 Firestore `leaderboard/{uid}` 한 문서만 업데이트되는지 확인.
5. 동일 기간에서 낮은 점수를 제출해도 기존 최고점이 유지되는지 확인.
6. 동일 기간에서 높은 점수를 제출하면 최고점이 갱신되는지 확인.
7. 오프라인 전환 시 랭킹 로드 실패 안내 문구가 표시되는지 확인.
8. 온라인 복귀 시 queued score flush 후 랭킹 반영 여부 확인.
9. Firestore Rules 게시 후 비인증 쓰기/삭제가 차단되는지 확인.

## 시각 체크
- 배경 이미지 표시
- 타일 이미지 표시
- 타일 이동 애니메이션 확인
- 합쳐질 때 흔들림/팝 이펙트 확인
- 합쳐질 때 점수 팝업(+점수) 표시
- 콤보 팝업이 중앙에 표시 후 사라짐
- 미니게임 타이머 진행바 표시
- 미니게임 히트/미스 이펙트 표시
- 보드/프레임/그리드 위치 정렬
- 배너 광고 영역 표시
- 랭킹 Top 1/2/3 배지 및 내 기록 카드 스타일 표시
